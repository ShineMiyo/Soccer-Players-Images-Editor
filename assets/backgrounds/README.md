
# Background Presets / 背景预设

## Folder Structure / 目录结构

Please organize your background images into the following subfolders:
请将您的背景图片分类放入以下子文件夹中：

1. **`assets/backgrounds/general/`** 
   - Place generic background images here (abstract, stadiums, patterns).
   - 放置通用背景图片（如抽象图、球场、纹理）。

2. **`assets/backgrounds/nations/`**
   - Place flag images here.
   - 放置国旗图片。

Supported formats: png, jpg, jpeg, webp, svg.
支持格式：png, jpg, jpeg, webp, svg.
